*A CY_BORG compatibility fork for itempiles auto hook by BragginRites.*
*CY_BORG for Foundry VTT can be found [here](https://foundryvtt.com/packages/cy-borg) and as well as the [GitHub](https://github.com/fvtt-fria-ligan/cyborg-foundry-3p).*

As according to the original repo, itempiles auto hook "turns npc tokens with <= 0 HP into item piles and turns them back into npc sheets if HP > 0."
